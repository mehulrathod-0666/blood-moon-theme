# Omarchy Color Mapping

Standard 16-color palette mapping for terminal consistency:

| Color | Name | Usual Use |
|-------|------|-----------|
| color0 | Black | Background / Deep Shadows |
| color1 | Red | Errors / Important Warnings |
| color2 | Green | Success / Strings |
| color3 | Yellow | Warnings / Functions |
| color4 | Blue | Keywords / Secondary UI |
| color5 | Magenta | Variables / Accents |
| color6 | Cyan | Types / Specialized text |
| color7 | White | Foreground / Regular text |
| color8 | Bright Black | Subtle UI / Comments |
| color9-15 | Bright Variants | High-contrast highlights |
