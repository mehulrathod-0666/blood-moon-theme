# Blood-Moon Theme

A deep, vibrant red and black theme for Omarchy, customized from the original pink-blood theme by ITSZXY.

## Features
- **Hyprland:** Custom borders and group styling.
- **Waybar:** Matching status bar styles.
- **Terminals:** Support for Alacritty, Kitty, Foot, and Ghostty.
- **Browser Themes:** High-contrast themes for Chrome and Firefox.
- **Other Apps:** Mako, SwayOSD, Walker, Neovim, VSCode, and Btop.

## Installation

### Main Theme
```bash
omarchy theme install https://github.com/mehulrathod-0666/omarchy-blood-moon-theme.git
```

### Browser Themes

#### Chrome / Chromium
1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select the `chrome-theme` folder inside your installed theme directory:
   `~/.config/omarchy/themes/blood-moon/chrome-theme`

#### Firefox
1. Copy the contents of `firefox-theme/userChrome.css` to your Firefox profile's `chrome` directory.
2. Enable `toolkit.legacyUserProfileCustomizations.stylesheets` in `about:config`.
