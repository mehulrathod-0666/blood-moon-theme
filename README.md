## 🎨 Features
- **Hyprland:** Vibrant Neon Purple active borders.
- **Waybar:** Cyber-dark status bar with Cyan accents.
- **Terminals:** Support for Alacritty, Kitty, and Ghostty in neon palette.
- **Browsers:** Automatic theming for Chrome and Firefox.

## 🚀 Installation
```bash
omarchy theme install https://github.com/mehulrathod-0666/omarchy-blood-moon-theme.git
```
