## 🎨 Features
- **Hyprland:** Emerald green active borders.
- **Waybar:** Forest dark status bar.
- **Terminals:** Support for Alacritty, Kitty, and Ghostty in green palette.
- **Browsers:** Automatic theming for Chrome and Firefox.

## 🚀 Installation
```bash
omarchy theme install https://github.com/mehulrathod-0666/omarchy-blood-moon-theme.git
```
